Debug - Teste de Carregamento de Produtos

Abra o console do navegador (F12) e navegue para:

  • Nova Receita
  • Editar Receita (Bolo de Chocolate)

Verifique se os logs de debug aparecem no console:

  • 🛒 [ReceitaForm] Produtos carregados
  • 🛒 [ReceitaForm] Quantidade de produtos
  • 🛒 [ReceitaIngredientes] Produtos recebidos
  • 🌐 [API] logs de requisições
